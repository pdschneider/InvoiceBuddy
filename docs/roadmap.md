
# 💸 Invoice Buddy Roadmap

This document outlines the vision, planned features, and development priorities for **Invoice Buddy**.

It is a living document and will be updated periodically. Some features may not be included in Invoice Buddy and other features not listed may be added.

## 🎯 Project Vision

Invoice Buddy is an invoice and receipt management app designed to simplify financial data entry by automating filenames, spreadsheet data entry, and archival.

## 📅 Current Status (April 2026)

- **Version**: 0.1.x (Stable release available)
- **Core Features**: Auto-generated filenames, quick spreadsheet entry, fast file archival
- **Platforms**: Linux & Windows 10/11

## 🗺️ Planned Milestones

The roadmap for Invoice Buddy includes several planned features prior to the official 1.0 release. Each minor release follows a general theme and is always subject to change.

## 0.2.x - Networking (Google Sheets)

Updates in the 0.2.x series will focus on adding support for Google Sheets to give users more spreadsheet options.

### Major Improvements
- [ ] Add initial support for Google Sheets
- [ ] Include support for OCR fallback on Windows
- [ ] Full Windows desktop integration

### Other Improvements
- [ ] Support writing to document while open
- [ ] Support individual archive paths for different types
- [ ] Improved data entry readability

## 0.3.x - Printer Support

This series of updates will focus on adding support for printers in order to save physical copies of invoices/receipts upon archival.

### Major Improvements
- [ ] Add support for printer selection

### Minor Improvements
- [ ] Keyboard shortcuts

## 0.4.x - Auto-Name

This update cycle will improve upon the existing auto-name feature to enhance accuracy and add more options.

### Major Improvements
- [ ] Add description (what was purchased)
- [ ] Add Terms (Net 30, Due Upon Receipt, etc.)
- [ ] Add Total Amount
- [ ] Add Total Tax
- [ ] Add Sales Tax
- [ ] Custom Field (user determined)

## 0.5.x - Initial User Experience

Updates in this cycle will enhance the initial user experience by allowing the creation of a spreadsheet file in-app and more.

### Major Improvements
- [ ] Feature in-app spreadsheet creation

## 0.6.x - Polish / Pre-Major Release

This is the final update cycle before the official release of v1.0.0. The focus here is on adding any final features and fixing bugs before the major release.

### Major Improvements
- [ ] Send with Note
- [ ] Progress bars for pdfsearch / data entry

## 📝 Notes

- Priorities may shift based on user feedback and bug reports.
- Feature requests are welcome — please open a GitHub issue.
- This roadmap focuses primarily on **user-facing** improvements. Technical debt and refactoring happen continuously.
